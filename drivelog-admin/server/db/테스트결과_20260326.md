- **수정**: company_id NULL 허용 + JOIN을 LEFT JOIN으로 변경
- **DB 마이그레이션**: `ALTER TABLE inquiries MODIFY COLUMN company_id INT NULL;`

## ❌ 버그 2: 운행등록 최소필드 500 → ✅ 수정완료
