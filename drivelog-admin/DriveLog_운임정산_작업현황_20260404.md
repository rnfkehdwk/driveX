## ⚠️ DB 마이그레이션 주의사항

`companies.company_id`가 **bigint(20)** 이라서 `settlement_groups`도 BIGINT로 통일해야 함:

```sql
CREATE TABLE IF NOT EXISTS settlement_groups (
  group_id     BIGINT AUTO_INCREMENT PRIMARY KEY,
  company_id   BIGINT NOT NULL,
  name         VARCHAR(50) NOT NULL,
  color        VARCHAR(20) DEFAULT '#64748b',
  sort_order   INT DEFAULT 0,
  created_at   TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at   TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  FOREIGN KEY (company_id) REFERENCES companies(company_id) ON DELETE CASCADE,
  UNIQUE KEY uk_company_name (company_id, name)
);

ALTER TABLE payment_types
  ADD COLUMN settlement_group_id BIGINT NULL,
  ADD CONSTRAINT fk_pt_settlement_group
    FOREIGN KEY (settlement_group_id) REFERENCES settlement_groups(group_id) ON DELETE SET NULL;

INSERT INTO settlement_groups (company_id, name, color, sort_order)
SELECT company_id, '기사 보유', '#d97706', 1 FROM companies
ON DUPLICATE KEY UPDATE name = name;

INSERT INTO settlement_groups (company_id, name, color, sort_order)
SELECT company_id, '회사 보유', '#0f6e56', 2 FROM companies
ON DUPLICATE KEY UPDATE name = name;
```

마이그레이션 실행:
```bash
sudo docker exec -i drivelog-db mariadb -uroot -p'Drivelog12!@' drivelog_db < /volume1/docker/drivelog/server/db/migration_settlement_groups.sql
```

## 🛠️ API 엔드포인트 정리

### 정산 그룹
- `GET    /api/settlement-groups` — 목록 조회
- `POST   /api/settlement-groups` — 등록 (name, color, sort_order, company_id)
- `PUT    /api/settlement-groups/:id` — 수정
- `DELETE /api/settlement-groups/:id` — 삭제

### 결제구분 (확장)
- `POST/PUT` 시 `settlement_group_id` 필드 추가됨
- `GET` 응답에 `settlement_group_name`, `settlement_group_color` 포함

### 일일 운임 정산
- `GET /api/settlements/daily?date=2026-04-04`
- 응답:
  ```json
  {
    "date": "2026-04-04",
    "total": { "fare": 850000, "count": 12 },
    "groups": [{ "group_id", "name", "color", "total", "count" }],
    "payments": [{ "code", "label", "group_id", "group_name", "group_color", "total", "count" }],
    "riders": [{ "rider_id", "rider_name", "total", "count", "groups": {gid: amount} }],
    "rides": [...상세 운행 내역]
  }
  ```
